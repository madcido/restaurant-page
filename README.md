## Installation Guide
### Prerequisities
System dependencies:
```
$ node -v
  v10.15.3

$ npm -v
  6.4.1

$ webpack -v
  4.34.0
```

#### Local setup
##### Clone project
```
$ git clone https://github.com/madcido/restaurant-page.git
$ cd ./restaurant-page
```
##### Install dependencies
```
$ npm install
```
##### Build project
```
$ npm start
```
